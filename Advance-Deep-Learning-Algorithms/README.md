# Advance Deep Learning Algorithms

This directory contains implementations and research on advanced deep learning algorithms.

## Topics Covered

- **Neural Architecture Search (NAS)** — Automated design of neural network architectures
- **Transformers & Attention Mechanisms** — Self-attention, multi-head attention, vision transformers
- **Generative Models** — GANs, VAEs, Diffusion Models, Flow-based models
- **Graph Neural Networks (GNNs)** — Node classification, link prediction, graph generation
- **Meta-Learning** — Few-shot learning, MAML, Prototypical Networks
- **Self-Supervised & Contrastive Learning** — SimCLR, BYOL, MoCo
- **Reinforcement Learning** — Deep Q-Networks, Policy Gradient, Actor-Critic methods
- **Neuro-Symbolic AI** — Combining neural networks with symbolic reasoning
