# High Performance Computing

This directory contains implementations and notes on high-performance computing concepts.

## Topics Covered

- **Parallel Programming** — OpenMP, MPI, Pthreads
- **GPU Computing** — CUDA, OpenCL, GPU architecture
- **Distributed Systems** — MapReduce, Spark, distributed memory models
- **Performance Optimization** — Cache optimization, vectorization, loop unrolling
- **HPC Architectures** — Clusters, supercomputers, memory hierarchies
- **Numerical Methods** — Matrix computations, FFT, linear algebra on HPC
- **Cloud & Container HPC** — Kubernetes, Docker for scientific computing
- **Benchmarking & Profiling** — Performance analysis, scalability testing
